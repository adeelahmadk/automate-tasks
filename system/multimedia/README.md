# Multimedia Processing

Scripts to automate tasks on multimedia files e.g. images, audios, videos.

| Script | Language | Category | Description |
| --- | --- | --- | --- |
| [`img_orient.sh`](./img_orient.sh) | bash | Multimedia | batch update orientation of photos (single dep.). |
| [`img_orientx.sh`](./img_orientx.sh) | bash | Multimedia | batch update orientation of photos efficiently. |
| [`img_orient_progress.sh`](./img_orient_progress.sh) | bash | Multimedia | batch process of photo metadata (orientation) with progress bar. |
